PS必会技能

photoshop是一个非常强大的图片处理工具，是设计师要精通的一项技能

PS技能的需求：切图、修图、测量

PS工具：
	移动工具
	矩形选框工具
	裁剪工具
	吸管工具
	横排文字工具
	手抓
	缩放
	标尺
	自由变换
快捷键
	ctrl    +      c        复制
	ctrl    +      v        粘贴
	ctrl    +      n        新建文件
	ctrl    +      s        保存

	空格键                 手抓工具
	ctrl    +      t        自由变形
	ctrl    +    加号     放大
	ctrl    +    减号     缩小
	ctrl    +      r        标尺
	ctrl    +     alt     +    shift   +   s    储存为web格式
	ctrl    +     alt     +    shift   +   e    盖印可见图层
图片格式 （ PSD | JPG/GIF/PNG）
	PSD （源文件）

	JPG/GIF/PNG  （导出图）

	JPG：不支持透明半透明，所有空白区域填充白色

	GIF：支持透明，不支持半透明

	png8：支持透明，不支持半透明

	png24：支持透明，也支持半透明
JPG/GIF/PNG的应用
	JPG：网页中的大图，高清图（体积大）；

	GIF：网页中的小图标（动画图片）；

	PNG8：网页中的小图标；

	PNG24：图像中存在半透明效果的图片；
练习：
	打开淘宝、天猫等网站，抓屏后保存为图片，然后还原为HTML页面。
